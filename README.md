# The infoRouter Ruby Gem

A Ruby interface to the infoRouter SOAP API

## Installation

Add this line to your application's Gemfile:

    gem 'inforouter'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install inforouter
